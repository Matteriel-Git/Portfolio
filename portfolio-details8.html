About

Real Estate website that provides an easy-to-use interface for viewing properties for rent or purchase. The website is mobile-responsive and easy to navigate.


  • - Shows property listings and uses tabs for different types of properties.
  • - Reviews from customers.
  • - User can get in touch with business thru email.
  • - User can request additional information about the properties by using the contact form.
